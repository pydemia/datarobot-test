Agent¶

The MLOps agent forwards buffered messages from the MLOps library to DataRobot MLOps.

When the agent is running, it will look for buffered messages in a the configured directory or message queuing system and forward them to the designated DataRobot MLOps. You will need to install and configure the agent as indicated below:

  • MLOps agent installation and usage
    • Unpack the MLOps Tar File
    • MLOps Tracking Agent
      • Update the configuration file
      • Run the MLOps agent
        • Start the agent using the config file
        • Alternatively, start the agent using environment variables

      • Shut down the agent
