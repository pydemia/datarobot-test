MLOps Client (7.2.1)¶

The MLOps client is a library that can be used to interact directly with DataRobot MLOps. It can be used to create deployments, upload training datasets, and more.

Because the MLOps client interacts with DataRobot MLOps, it must be run from an environment with connectivity to DataRobot MLOps.

create_model_package(model_info)[source]¶

Create an external model package in DataRobot MLOps from JSON configuration.

Parameters:model_info (dict) – a JSON object of model parameters
Returns:model package ID of newly created model
Return type:str
Raises:DRConnectedException – if model package creation failed.

Example JSON for a regression model:

{
  "name": "Lending club regression",
  "modelDescription": {
    "description": "Regression on lending club dataset"
  }
  "target": {
    "type": "Regression",
    "name": "loan_amnt
  }
}

Example JSON for a binary classification model:

{
  "name": "Surgical Model",
  "modelDescription": {
    "description": "Binary classification on surgical dataset",
    "location": "/tmp/myModel"
  },
  "target": {
     "type": "Binary",
     "name": "complication",
     "classNames": ["Yes","No"],  # minority/positive class should be listed first
     "predictionThreshold": 0.5
    }
}

Example JSON for a multiclass classification model:

{
    "name": "Iris classifier",
    "modelDescription": {
        "description": "Classification on iris dataset",
        "location": "/tmp/myModel"
    },
    "target": {
        "type": "Multiclass",
        "name": "Species",
        "classNames": [
            "Iris-versicolor",
            "Iris-virginica",
            "Iris-setosa"
        ]
    }
}
